# 🧮 小学六年级数学计算卷生成器 (Grade 6 Math Paper Generator)

![License](https://img.shields.io/badge/license-MIT-blue.svg) ![Language](https://img.shields.io/badge/language-HTML%2FCSS%2FJS-yellow.svg) ![Status](https://img.shields.io/badge/status-Stable-green.svg)

一个轻量级、零依赖的网页应用，用于自动生成小学六年级（及初一衔接）的数学计算专项练习题。基于浏览器运行，支持一键生成 A4 打印版试卷，内置智能出题算法，确保题目逻辑严谨且答案“干净”（整数或有限小数）。

## ✨ 主要功能

* **🖨️ A4 完美打印**：专为打印设计的 CSS 布局，符合标准 A4 纸张排版，字体清晰（楷体/Times New Roman）。
* **♾️ 无限题库**：完全基于算法实时生成，每次刷新都是一套全新的试卷。
* **👁️ 答案显隐**：一键切换显示/隐藏参考答案，方便家长批改或学生自测。
* **🧩 智能算法**：
    * 自动规避无意义题目（如 `1 × 1`，或结果为无限循环小数的情况）。
    * 分数/小数/百分数混合运算逻辑。
* **⚙️ 自定义设置**：可选择题目难度（基础/提高）及包含的知识点（分数乘除、百分数）。

## 📝 题型概览

本系统包含四大核心计算板块：

### 1. 口算/直接写得数
* 包含分数加减乘除、百分数运算。
* **算法优化**：自动剔除“整数对整数”的简单运算，强制生成分数运算题。

### 2. 脱式计算（简便运算）
包含六年级核心简算模型，并**严格过滤**出现系数为 `0` 或 `1` 的无效题目：
* **连乘约分**：确保三个分数相乘能进行交叉约分。
* **乘法分配律**：`a × c + b × c` 模型。
* **括号拆分**：`m × (a + b)` 模型。
* **提取公因数**：`(a × f + b × f)`，结果凑整。
* **小数分数互化**：如 `3.5 × 1/2 + ...`。

### 3. 单位换算
* 覆盖长度、重量、面积、体积、容积。
* 支持“大单位变小单位”和“小单位变大单位”的进率换算。

### 4. 解方程（六大进阶模型）
针对小升初重点难点设计的 6 种方程模型：
1.  **同类项合并型**：`x - 40%x = 120`
2.  **解比例型**：`10 : x = 2/3 : 4/5`（内项积=外项积）
3.  **去括号型**：`3(x + 1) = 2x + 5`
4.  **去分母型**：`(x)/2 - 1 = (x+2)/6 + 2`（确保常数项能整除）
5.  **三数混合型**：分数、小数、百分数混合系数。
6.  **先算后解型**：`4 × 3.5 - 2x = 10.2`

## 🚀 快速开始

本项目是一个单文件应用（Single File Application），**无需安装 Node.js，无需配置数据库**。

### 方法一：直接运行
1. 下载本项目中的 `index.html` 文件。
2. 双击直接在浏览器（推荐 Chrome 或 Edge）中打开。
3. 点击左侧侧边栏的“🔄 刷新题目”生成试卷。
4. 点击“📄 打印 / 下载PDF”即可输出。

### 方法二：部署到 GitHub Pages
如果你想让更多人使用，可以直接将代码上传到 GitHub 仓库，并开启 GitHub Pages 服务，即可获得一个在线访问链接。

## 🛠️ 技术栈

* **HTML5**: 语义化标签构建结构。
* **CSS3**: 
    * 使用 `Grid` 布局实现复杂的题目排版（2列/3列/4列）。
    * 使用 `@media print` 媒体查询实现打印时的去UI化和样式控制。
* **JavaScript (ES6+)**:
    * 面向对象编程（`Fraction` 类）处理分数运算、约分、输出 HTML 格式。
    * 复杂的数学逻辑生成与校验算法。
